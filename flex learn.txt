/***** flex-wrap *****/
flex-wrap:
    - nowrap; (по умолчанию): элементы стараются вместиться в одну строку, сжимаясь при необходимости.
    - wrap;: элементы переносятся на следующую строку, если не помещаются.
    - wrap-reverse;: элементы также переносятся, но в обратном порядке.

для корректного переноса элементов
* {
  box-sizing: border-box;
}
Включение box-sizing: border-box позволяет учитывать паддинги и бордеры в общей ширине элементов, что облегчает создание точной сетки.
.item {
    width: calc(100% / 3)
}

.container {
    flex-warp: wrap-reverse;
}
